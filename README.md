# Stone Paper Scissor Game
This is a simple Python project where the user plays Stone Paper Scissor against the computer.
The computer randomly chooses Stone, Paper, or Scissor, and the program decides who wins.

## Features
* User vs Computer game
* Random computer choice
* Shows winner, loser, or draw
* Beginner-friendly Python project
* Easy to understand code

## How to Run
1. Install Python
2. Download or clone this repository
3. Run the Python file

```bash
python main.py
```
